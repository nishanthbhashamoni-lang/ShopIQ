# ShopIQ 🚀

ShopIQ is a smart shopping assistant that helps users compare products across multiple e-commerce platforms using a single product link.

## 🔍 Problem
Users manually compare products across different shopping platforms.
This process is time-consuming and inefficient.

## 💡 Solution
ShopIQ allows users to:
- Paste a product link
- Automatically extract product name
- Generate search links for multiple platforms
- Compare easily

## 🛠 Tech Stack
- Python
- Flask
- REST API Architecture

## ▶ How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the server:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
